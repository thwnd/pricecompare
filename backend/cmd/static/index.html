가격비교마스터 API 문서

이 페이지는 가격비교마스터의 백엔드 API 문서입니다. 프론트엔드 웹사이트는 여기에서 확인하세요.

API 엔드포인트

GET /api/items

전체 아이템 목록을 페이지네이션과 함께 반환합니다.

쿼리 파라미터:

  • page: 페이지 번호 (기본값: 1)
  • limit: 페이지당 아이템 수 (기본값: 20)

예시: /api/items?page=1&limit=10

GET /api/items/:category

지정된 카테고리의 아이템 목록을 반환합니다.

쿼리 파라미터:

  • page: 페이지 번호 (기본값: 1)
  • limit: 페이지당 아이템 수 (기본값: 20)

예시: /api/items/books

GET /api/item/:id

ID로 개별 아이템 조회

예시: /api/item/123

GET /api/search

아이템 검색 API

쿼리 파라미터:

  • q: 검색어 (필수)
  • page: 페이지 번호 (기본값: 1)
  • limit: 페이지당 아이템 수 (기본값: 20)

예시: /api/search?q=book
